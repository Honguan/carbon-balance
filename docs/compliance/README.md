# 法規與標準工程邊界

本目錄只保存工程所需摘要與證據鏈，不保存受著作權保護的完整 ISO 文件，也不取代法律、PCR 或查驗專業意見。

權威順序：

1. 現行法律、主管機關規則與已確認有效的 PCR。
2. ISO 14067、ISO 14040、ISO 14044、ISO/TS 14027 適用要求摘要。
3. 已接受 ADR。
4. PRD、資料模型、架構與驗收文件。
5. Issue 與驗收標準。
6. 歷史報告、試算表、SQL 與 PHP。

欠缺 PCR 有效性、係數授權或領域確認時，資料只能保留為 historical、quarantined 或 pending-domain-review。
